# contact_survey_hk
Social contact survey in HK

## Overview

This repository contains data and R scripts for analyzing social contact patterns in Hong Kong. The analysis examines contact patterns across different demographics, locations, and contact types to understand social mixing behavior.

## Repository Contents

- **`data/`** - Survey data
  - `raw/` - Original contact survey data
  - `processed/` - Cleaned and processed data (generated by scripts)
  
- **`scripts/`** - R analysis scripts
  - `01_data_preprocessing.R` - Data cleaning and preprocessing
  - `02_data_analysis.R` - Statistical analysis
  - `03_visualization.R` - Data visualization
  - `run_all_analyses.R` - Master script to run complete pipeline
  
- **`output/`** - Analysis outputs (generated by scripts)
  - `figures/` - Visualizations (PNG files)
  - `tables/` - Analysis results (CSV files)

## Quick Start

To run the complete analysis:

```r
source("scripts/run_all_analyses.R")
```

For detailed documentation on the data analysis, see [DATA_ANALYSIS_README.md](DATA_ANALYSIS_README.md).

## Requirements

- R version 4.0.0 or higher
- Required packages: `dplyr`, `readr`, `ggplot2`, `tidyr`, `lubridate`, `here`
